# Rozgar Map Quiz

Quiz portal with Vercel backend + Netlify frontend + MongoDB Atlas.

## Deploy
See DEPLOY.md for step-by-step instructions.

## Local Development
```bash
cd server && npm install && npm run dev    # Backend: http://localhost:5000
cd client && npm install && npm run dev    # Frontend: http://localhost:3000
```

## Structure
```
rozgar-map-quiz/
├── client/          # React frontend (Netlify)
├── server/          # Node.js API (Vercel)
└── DEPLOY.md        # Deployment guide
```
